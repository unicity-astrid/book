# The Astrid Book

The canonical reference for Astrid OS: the kernel, the capsule model, the host ABI, the bus, and the security model.

Built with [mdBook](https://github.com/rust-lang/mdBook).

```bash
mdbook serve --open
```

The reference appendices (capability catalog, host ABI error codes, topic registry) are generated from the Astrid source by `tools/gen-appendices.sh`. The cross-chapter "See also" footers are generated by `tools/gen-see-also.pl`. Both read the polyrepo source tree, so run them from a checkout that has `core/`, `wit/`, and `capsules/` as siblings.
